AppNameToClipboard
A tiny app that copies its name to the clipboard.

Download AppNameToClipboard
That's all it does. When you click it, it will copy the filename it currently has to the Windows clipboard, minus the '.exe' on the end, which you may not be able to see unless you have changed a Windows setting.

It doesn't have any display at all, no windows or dialogues. Not even an icon, because it too is too cheap.
Here it is being clicked, with a long name for demonstration.

It won't show anything at all, and will immediately close again.

Pasting into an empty notepad. Ctrl-V works too.
And this is the result, as expected.

To use it just change the name of the app, which is usually done by selecting it with the mouse and then pressing F2.
Or by right clicking the mouse and selecting 'rename'

There are some characters you can't use in filenames, Windows will tell you.

Use it to store things you type often.

Probably should not use it for passwords though.
